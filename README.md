# BONK! 🏓

A Pong game made with **C++** and **Raylib**.

This is my first game project while learning game development. I built it to practice the basics of making a game from scratch, including game loops, player controls, collision detection, rendering, audio, and object-oriented programming.

## 🎮 About The Game

BONK! is a classic Pong remake where you play against a CPU paddle.

The goal is simple:
- Move your paddle
- Hit the ball
- Score points against the CPU

## ✨ Features

- Player-controlled paddle
- CPU opponent paddle
- Ball physics
- Collision detection
- Score system
- Background music
- Sound effects
- 60 FPS gameplay
- Simple object-oriented C++ structure

## 🛠️ Built With

- **C++**
- **Raylib**

## 🎯 What I Learned

While making this project, I practiced:

- Creating a game loop
- Drawing shapes and objects with Raylib
- Handling keyboard input
- Using classes in C++
- Managing game objects
- Collision detection
- Playing sounds and music
- Organizing a small game project

## 🚀 How To Run

### Requirements

You need:
- A C++ compiler
- Raylib installed

### Steps

1. Clone this repository:

```bash
git clone https://github.com/TheRedCreator/Pong.git
```

Open the project in your C++ editor
Make sure the Resources folder is in the correct location:

```
Pong/
│
├── main.cpp
├── Resources/
│   ├── Music.wav
│   ├── Win.wav
│   ├── Fail.wav
│   └── Bounce.wav
│
└── README.md

```
Compile and run the game
```
🎮 Controls
Key	Action
↑ Up Arrow	Move paddle up
↓ Down Arrow	Move paddle down
```
